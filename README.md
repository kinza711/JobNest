**JobNest – Job Portal Platform**

JobNest is a full-stack Job Portal application built with the MERN stack that connects job seekers with recruiters through a structured recruitment workflow. The platform supports role-based access for Admin, HR, and Job Seekers, enabling job posting, application management, and candidate tracking.

The system also includes resume upload and preview functionality along with automated email notifications that keep applicants informed about their application submission and status updates.

**Key Features**

**Job Seeker**

Browse and search available job postings

View job details and apply for positions

Upload resumes while applying

Track application status

Receive automated email confirmation after applying

Receive email notifications when application status is updated

Manage profile and view submitted applications

Preview and download uploaded resumes

**HR / Recruiter**

Create, update, and delete job postings

Open or close job listings

View candidate applications

Preview and download applicant resumes

Update application status (Shortlisted, Rejected, Hired)

Manage job seeker accounts

Maintain recruiter profile

**Admin**

Full access to HR features

Manage HR accounts and job seekers

Monitor job postings and applications
